# 静态链表说明

使用数组来实现链式存储结构，目的是方便在不设指针类型的高级程序设计语言中使用链式结构

c语言定义数据结构

    #define MAX_SIZE 1000

    // 所有的类型都统一定义为ElemType
    typedef int ElemType;

    typedef struct
    {
        ElemType data;
        int cur;
    } component, LinkList[MAX_SIZE];
